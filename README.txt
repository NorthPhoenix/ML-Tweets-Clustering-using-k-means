Option 1 - Run KMeans.ipynb file in Google Colab
    1. Go to Google Colab.
    2. Import KMeans.ipynb into an empty notebook.
    3. Run all cells.
    4. Output and graphs are inline with code.



Option 2 - Run KMeans.ipynb file with VSCode
    1. Install Python ( Tested on Python 3.11.2 )
    
    2. Install required libraries: 
    numpy:
        pip install numpy
    pandas:
        pip install pandas
    matplotlib:
        python -m pip install -U pip
        python -m pip install -U matplotlib
    sklearn:
        pip install -U scikit-learn

    3. Install VSCode.

    4. Install the official Python extension for VSCode.

    5. Open KMeans.ipynb with VSCode.

    6. Run all cells.
